üíª Powerful Features

Full CRUD Operations

Complete support to create, read, update, and delete books and authors, providing a comprehensive management system.

Advanced Querying

Fetch data with fine-grained control using pagination, multi-field sorting, and dynamic filtering by title or genre.

Interactive API Docs

Auto-generated Swagger UI allows for easy, live testing and clear documentation of all API endpoints without any setup.

Layered Architecture

Built with a clean separation of concerns (Controller, Service, Repository) for enhanced maintainability and scalability.

In-Memory Database

Utilizes H2 for zero-configuration, rapid development cycles. The database automatically resets on each launch.

Robust Error Handling

Provides clear, specific HTTP error messages for invalid inputs, missing resources, and server-side issues.
